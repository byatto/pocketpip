# Changelog

All notable changes to littlemoments will be documented in this file.

## [1.0.0] - 2026-02-13

### Added
- Initial release
- Capture view with instant text input
- Quick-tap categories: Quote, First, Funny, Note
- Date override for backdating moments
- Freeform tag system with autocomplete
- Review inbox for uncategorised moments
- Timeline view with chronological grouping
- Search and filter by category, tag, and text
- Pin favourite moments
- Cloud sync via Google Apps Script
- Dark mode with system-aware toggle
- Keyboard shortcuts for all core actions
- Export/Import JSON
- Soft delete with 30-day trash and undo
- Responsive mobile-first design
